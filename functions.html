  1. Working with parabolas or quadratic functions: Given the equation
    y = x2 + 6x + 5
  2. First, find the y-intercept.
    3. The y intercept is a poing where the function touches the y-axis of the coordinate or cartesian plane, therefore the value of x is 0.
    Hence, to find y-intercept , you simply put zero where there is x in the given equation.
    Making the number without x (C) in the equation always the y-intercept.
    y = x2 + 6x + 5
    y = 02 + 6(0) + 5
    y = 0 + 0 + 5
    y = 5
  3. Then, find the x-intercept or roots.
    4. The x-intercept is a poing where the function touches the x-axis of the coordinate or cartesian plane, therefore the value of y is 0. Hence, to find x-intercept , you simply put zero where there is y in the given equation and solve for x.
    y = x2 + 6x + 5
    0 = x2 + 6x + 5
    0 = x2 + 5x + x + 5
    0 = (x2 + 5x) (+ x + 5)
    x(x + 5) 1(x + 5)
    x + 1 = 0, x + 5 = 0
    Therefore x = -1 or x = -5
    If you still struggling with solve for x click the button below
    Algebra
  4. Find the vertex or turning point:
    5. Find the x-coordinate of the vertex by using x = -b/2a equation:
    x = -b/2a
    x = -(6)/2(1)
    x = -6/2
    x = -3
    Then find the y-coordinate of the vertex by substituting x-coordinate in to the given equation:
    y = (-3)2 + 6(-3) + 5
    y = 9 - 18 + 5
    y = - 9 + 5
    y = -4
    Therefore the vertex or turning point coordinates are (-3; -4)

Introduction to hyperbola graphing

×

  1. Hyperbolas

  2. Have a general equation of f(x) = a/x + p + q
  3. if a is positive, you get a top-right and bottom-left graph.
  4. When a is negative, you get a top-left and bottom-right graph.
  5. x + p, gives the vertical asymptote by making x + p = 0 and solve for x, therefore vertical asymptope is x = -p
  6. q is the horizontal asymptote

    Steps when sketching a hyperbola

  1. find the asymptotes, e.g. f(x) = 3/x + 1 - 2
    Vertical asymptote is:
    x + 1 = 0
    x = -1
    Vertical asymptotes coordinates are (-1; 0)
    Horizontal asymptote is q. Therefore, vertical asymptotes coordinates are (0; -2)
  2. Predict the shape
    Since a is positive, the shape of the graph, should be top-right and bottom-left.
  3. Find the intercepts:
    y-intercept, x = 0
    f(0) = 3/0 + 1 - 2
    f(0) = 3/1 -2
    f(0) = 3 - 2
    f(0) = 1
    Y-intercepts cordinates are (0; 1)
    x- intercept/s, y = 0. remember f(x), h(x), g(x), etc., is y.
    0 = 3/x + 1 - 2
    2 = 3/x + 1
    2(x + 1) = 3
    2x + 2 = 3
    2x = 3 - 2
    2x = 1
    x = 1/2
    X-intercepts cordinates are (1/2; 0)
