    Statistics

    involves the collection, organization, analysis, interpretation, and presentation of a data set.
    It is used to help governments and companies make informed decision, and we hope after finishing the statistics section, you will start using stats also when making important decisions, especially.

  1. Data Sets

    Is a collection of related information that is usually represented as numbers.
    e.g. Heights of students in a given classroom.

  2. There are three basic different properties used to represent data in statistics.

  3. Mean, sometimes referred to as Average.

    is the value found by adding all numbers in the data set and then dividing by the number of values in the set.

    The equation is therefore, x = ∑/n

    ∑ is the sum of the values in a data set
    n is the number of values in a data set.
    e.g. Heights of ten male students in a class were measure in centimeters and the values were
    175, 166, 181, 164, 181, 171, 167, 182, 172, 176.
    Our recommended steps when working with a data set.
    1. Firstly, arrange your data set in an ascending order, meaning from lowest to highest before doing anything else. ,
      164, 166, 167, 171, 172, 175, 176, 181, 181, 182.
    2. Then, substitute the value into the equation to find the mean
      x = 164 + 166 + 167 + 171 + 172 + 175 + 176 + 181 + 181 + 182/10
      x = 1735/10
      x = 173.5
      Therefore, the average height of these 10 students is 173,5.

  4. Median

    The middle value of the data set.
    if the data set contains 2, 4, 6, 8, 10, 12, etc. values Median = mean of the two middle values. e.g. using the above data of 10 male students. 164, 166, 167, 171, 172, 175, 176, 181, 181, 182.
    Median = 172 + 175/2
    Median = 173.5
    However, when data set contains 3, 5, 7, 9, etc. values you do not need to do any calculation.
    e.g using the data above but instead of 10 students, we going to use 9
    164, 166, 167, 171, 172, 175, 176, 181, 181.
    In this scenario, the median is simply 172, because 172 is the exact middle number of the data set

  5. Mode

    The value in a data set that occurs more often.
    e.g. using the above data of 10 male students. 164, 166, 167, 171, 172, 175, 176, 181, 181, 182.
    The mode of this data is 181.

    Standard deviation

  1. Is the average difference of the data values from the mean.
  2. A low standard deviation indicates that the values tend to be close to the mean of the set, while a high standard deviation indicates that the values are spread out over a wider range.
  3. The equation standard deviation is s = √∑(Xi - mean)2/ n - 1
    Xi = value of each data set.

  1. Introducing additional properties of the data set

    • Range
    • Quartiles
    • Interquartile range
    • Outliers

  2. Range

    Is the difference between the highest value and the lowest value of the data set.
    E.g. 2, 4, 6, 8, 10, 12, 14, 16.
    Range = max - min
    Range = 16 - 2
    Range = 14

  3. Quartiles

    There are three quartiles

    • Second quartile (Q2)

      hich is the median of data set. Therefore, the median is the second quartile and the second quartile is the median.
      So what is the second quartile of the data set above?
      Median or Q2 = 10 + 8/2
      Median or Q2 = 9
      Thus second quartile of the data set above is 9.

    • First Quartile (Q1)

      is the median of the values in the data set that lower than the median of the whole data set.
      Therefore Q1 of the data set above is the median of 2, 4, 6, 8.
      Q1 = 6 + 4/ 2
      Q1 = 10/2
      Q1 = 5

    • Quartile 3 (Q3)

      is the median of the values in the data set that are higher than the median of whole data set.
      Hence, Q3 of the data set above is the median 10, 12, 14, 16.
      Q3 = 12 + 14/ 2
      Q3 = 26/2
      Q3 = 13

    • Interquartile range (IQR)

      This is the difference between Q3 and Q1
      IQR = Q3 - Q1
      = 13 - 5
      = 8

    • Outliers

      Any values of the data set that falls outside Q1 - 1.5 x IQR or Q3 + 1.5 x IQR.
      Hence, does our above data set have outliers?
      Q1 - 1.5 x IQR
      5 - 1.5 x 8
      5 - 12
      -7
      Q3 + 1.5 x IQR
      13 + 1.5 x 8
      13 + 12
      25
      Therefore, the above data set have no outliers because all the values fall within -7 and 25.

    INTRODUCTION TO GRAPHING YOUR DATA.

  1. Histogram

    A histogram is a graph used to represent the frequency distribution of a few data points of one variable.
    It is used to summarize discrete or continuous data that are measured on an interval scale. It is often used to illustrate the major features of the distribution of the data in a convenient form. It is also useful when dealing with large data sets (greater than 100 observations).

  2. Bar Graph

    Bar graphs are useful to compare things between different groups or to track changes over time. However, when trying to measure change over time, bar graphs are best when the changes are larger.

  3. Line Graphs

    Line graphs are useful to track changes over short and long periods of time. When smaller changes exist, line graphs are better to use than bar graphs.

  4. Pie Graphs

    Pie charts can be useful to show percentages of a whole, and represents percentages at a set point in time. Unlike bar graphs and line graphs, pie charts do not show changes over time.
