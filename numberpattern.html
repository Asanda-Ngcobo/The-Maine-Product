What is the difference, sum, product and quotient/ratio

To understand the difference between a linear sequence, quadratic sequence, geometric sequence and series, one needs to know what is the sum, product, difference and quotient/ratio.

arithmetic sequence

An arithmetic progression or arithmetic sequence is a sequence of numbers such that the difference between the consecutive terms is constant. And it is the most basic type of all sequences.

  2. {-5; -3; -1; 1;...}
  3. Tn = -5 + 2(5 - 1)

    Ma = -2/2

Quadratic number pattern

A quadratic sequence is a sequence of numbers in which the second difference between any two consecutive terms is constant.

×
  1. Finding the formula of a quadratic sequence:
    Tn = an2 + bn + c
  2. -145; -122; -101; -82;...
  3. First, you find the first differences
    -122 - (-145) = -122 + 145 = 23
    -101 - (-122) = -101 + 122 = 21
    -82 - (-101) = -82 + 101 = 19.
  4. Then, find the difference of the first differences:
    21 - 23 = -2
    19 - 21 = -2
  5. Find a by 2a = the 2nd difference
    2a = -2
    2a/2 = -2/2
    Therefore, a = -1
  6. Find b by 3a + b = the first value of the first differences
    3a + b = 23
    3(-1) + b = 23
    -3 + b = 23
    b = 23 + 3
    b = 26
  7. Last but not least, find c by a + b + c = the first value of the sequence
    a + b + c = -145
    -1 + 26 + c = -144
    25 + c = -145
    c = -170
  8. Therefore, the quadratic equation for this sequence is:
    -n2 + 26n - 170

Geometric sequences and series

Understand what is a geometric sequence, geometric series, finding their nth terms, finding the arithmetic and geometric mean.

×
  1. Geometric sequence defined by having a common ratio:
  2. Has an equation of an = a1(r)n - 1
    Where:
    r is the common ratio.
    n is the nth term.
    a1 is the first value of the sequence.
    an is the value of the nth term.
    E.g. 3; 6; 12; 24; 48; ....
  3. Finding the 6th term of the sequence above.
    Firstly, find the common ratio: 6/3 = 2; 12/6 = 2; 24/12 = 2; 48/12 = 2
    Therefore the common ration is 2.
    an = a1(r)n - 1
    a(6) = 3(2)6 - 1
    a(6) = 3(2)5
    a(6) = 3(32)
    a(6) = 96
    Therefore, the 6th term is 92.
  4. Geometry series is the sum of a geometric sequence:
  5. The equation for a finite series is:
    Sn = a1[1 - rn]/ 1 - r
  6. Finding the sum of the sequence above if it was finite
    3; 6; 12; 24; 48.
    Sn = a1[1 - rn]/ 1 - r
    Sn = 3[1 - 25]/ 1 - 2
    Sn = 3[1 - 32]/ 1 - 2
    Sn = 3[-31]/ - 1
    Sn = [-93]/ - 1
    Sn = 93
  7. However, if we made this sequence back to infinite (3; 6; 12; 24; 48;...), you couldn't find the sum. Because the sum of an infinite geometric sequence can only be found if the common ratio is less than 1 or greater than -1.
  8. If the ratio of an infinite geometric sequence is between -1 and 1, then you can use:
    S∞ = a1/ 1 - r
  9. Finding the geometric sequence mean:
    MG = √afirst-term x alast-term
  10. E.g. 3; 6; 12; 24; 48. MG = √a1 x a1
    MG = √3 x 48
    MG = √144
    MG = 12
