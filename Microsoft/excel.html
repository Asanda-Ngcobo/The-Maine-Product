  1. Simple Copy and Paste≈ 1:13 - ≈ 10:13

    2. There are different kinds of paste

    Paste (P) -everything in the source cell or range—data, formatting, formulas, validation, comments—is pasted to the destination cell(s).


    Value (V) - Lets suppose that the cell displays value 6, but, in fact, this is the formula (=2+1+3). With this type of pasting the value 6, not the formula is copied.


    Formulas (F) - In this case, Excel remembers that the pasted data has to be treated as a formula. Sometimes when you open more than one instance of Excel and then you use the standard paste method (Ctrl + V), it might be treated as an ordinary value. Therefore, in such cases, it is safer to use Copy as Formula.


    Transpose (T) - useful when you want to switch columns and rows.

  2. Using flash fill ≈ 10:20 - ≈ 15:31

    3. When you're entering data that has a pattern, like separating a full name into first and last names, you can use the Flash Fill feature in Excel for the web to save time and have Excel fill in the data for you.

  3. Creating a directory or table of content ≈ 19:18 - End

    Useful when you have too many worksheets on one workbook. You can create one worksheet specifically as your table of content, that would link you to your every worksheet. making easy to access all your worksheets.
